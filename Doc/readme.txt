【*】程序简介

-工程名称		：流水灯
-实验平台  		：野火 STM32F103C8T6-STM32开发板  

【 ！】功能简介：

控制GPIO引脚输出高低电平控制LED灯的亮灭，实现点灯的效果


【 ！】实验操作：

	1.编译并下载程序到开发板即可观察现象；
	2.可以观察到红、绿、蓝灯依次点亮然后再一起闪灭。

/*********************************************************************************************/

【*】 引脚分配

LED_R->LED1->PA1	核心板板载LED灯低电平亮 
LED_G->LED2->PA2	核心板板载LED灯低电平亮 
LED_B->LED3->PA3	核心板板载LED灯低电平亮

/*********************************************************************************************/

【*】程序描述：

< main.c >

	1.初始化LED对应的GPIO，初始化LED端口
	2.控制LED的亮灭
	3.依次点亮红、绿、蓝灯
	4.所有灯同时闪
